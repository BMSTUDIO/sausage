LazyPages
===========

A contextual pagination jQuery plugin for infinite scrolling pages. Early days--pretty rough at the moment. Check the demo!

Demos
--------------

- [Basic](http://christophercliff.github.com/lazypages/demos/basic.html "Basic")


Requirements
-------------

- jQuery 1.4.3
- jQuery UI 1.8.5
- jQuery Templates
